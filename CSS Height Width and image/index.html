CSS Height, width and max-width

The CSS height and width properties are used to set the height and width of an element

setting hieght and width

the height and width properties do not include padding, borders, or margins.
It sets the height/width of the area inside the padding,border and margin of the element.

css height and width values

the height and width properties may have the following values:

  • auto - this is default. the browser calculates the height and width

  • length - defines the height/width in px, cm, etc.

  • % - defines the height/width in percent of the containing block

  • intial - sets the height/width to its default value

  • inherit - the height/width values inherited from its parcent value.

example :

set the height and width of a <.div> element
div{height:200px; width: 50%;}

style images using CSS (image filters)

the css filter property adds visual effects to an element:

  • brightness-adjust the brightness of an image

  • contrast- adjust the contrast of an image

  • grayscale- converts an image to grayscale

  • sepia - gives an image a sepia tone

  • hue- rotate - rotate the colors of an image

  • saturate - adjust the saturation of an image

  • invert- inverts the color of an image

  • opacity - adjust the opacity of an image

  • blur- adds a blur effect to an iamge

    • note: the filter property is not supported in internet explorer or edge 12
